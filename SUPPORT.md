# Support

For help with a Mara public repository:

- Use the repository README first.
- Search existing issues before opening a new one.
- Open a bug report for reproducible defects.
- Open a feature request for product or API improvements.
- Open a docs issue for unclear or missing documentation.

For private, account, billing, security, or business questions, contact `contact@iammara.com`.

Do not post secrets, real customer data, private logs, access tokens, API keys, or real personal data
in public issues or pull requests.
